## Basic container deploy ##
- kubectl create -f [pod-specification].yaml
- [pod-specification].yaml
		apiVersion: v1
		kind: Pod
		metadata:
		  name: shell-demo
		spec:
		  volumes:
		  - name: shared-data
			emptyDir: {}
		  containers:
		  - name: nginx
			image: nginx
			volumeMounts:
			- name: shared-data
			  mountPath: /usr/share/nginx/html
- kubectl get pod [pod]	
- kubectl exec -it [pod] -- /bin/bash
